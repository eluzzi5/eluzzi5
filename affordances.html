Predicting Object Affordances

Internship Project, Tufts Human Robot Interaction Lab - 2016-2017

Skills Used

  • Research

Tools Used

  • PHP
  • HTML
  • JavaScript
  • Amazon Mechanical Turk

This project sought to develop a new model of object affordance perception for robotic agents. Broadly, the affordances of an object are those functions or effects which it offers an agent. As humans, we are able to come up with solutions by utilizing objects in creative ways. For example, we are able to perceive the ability of a mug not only to hold a beverage, but also to act as a paperweight or doorstop. Inspired by this, we sought to define affordances in a way such that robotic agents operating according to this definition could perceive object affordances in a way that was not only accurate, but creative.


Model

The definition we came up with was similar to that of Erol Sahin et al., who are concerned with not only what can be afforded by an object but what should be. Their definition considers not only the physical properties of an object but also the contextual and environmental information that influence higher-level cognitive perception of the object. For example, rather than an agent perceiving the affordance of “sitability” – the ability to be sat upon – for all chairs, an agent operating under this definition would consider contextual factors such as who owns the chair when considering its sitability. In this model, an agent learns about the effects of an action on an object through trial and error.


Our definition sought to modify the model proposed by Sahin et al. in order to allow for more creative projection. Rather than a knowledge base developed through trial and error, our agent would have a set of rules which are referred to in order to predict object affordances. We define an affordance as follows:
An affordance a is a relation between an action m and a feature f, that is a normatively permissible action possibility m on a perceivable feature f. Action possibility m is an action in the set of available actions M for an agent X. Available actions adhere to the societal and contextual norms of the environment, which has a set of rules R.


Online Study

To learn more about the ways in which humans perceive object affordances, we ran a study observing people’s reasoning about objects based in different contexts. Each question consisted of a short description of the context and four objects. The participant was asked to select which objects could be used to accomplish the implied goal in the given context. For each question, one of two contexts was randomly selected. This selection was counterbalanced among participants. The objects that a participant could select varied in their functionality. Some were straightforward in the sense that they are commonly used to accomplish the desired goal. Others were more subtle in that they could potentially be used to accomplish the desired goal, though it is not a typical use case for the object.


This was an online study through Amazon Mechanical Turk, and I wrote the study pages using JavaScript, HTML, and PHP. For each question, a rule was considered in relation to certain affordances and features of an object.


Ex.
rule: water conforms to the shape of its container
affordances: hold water, contain
features: walls, container, closed bottom

Definition specifics

In order to illustrate the commonalities and differences between our and Sahin et al.’s definitions, we provide three examples.


Example 1: Both definitions correctly predict affordance
Touchability of fire
Our model: The agent will know that fire does not afford touching based on the set of normative rules R. It can predict the existence or lack of affordances using these rules. In this situation, even though the agent may not have previously observed the effect, it would be able to predict that the fire does not afford touching.
Sahin's model: The agent will learn the effect through observing objects (such as kindling) burn in the fire. Because the agent is aware of the effect, it has acquired the knowledge that the fire does not afford touching.


Example 2: Our definition correctly predicts affordance
Balanceability of an unfamiliar box
Our model: the agent has never acted upon the box before, yet based on the agent’s ability to balance similar objects in the past, in relation with the box’s property of having a flat surface, the agent can predict that the box is able to be balanced.
Sahin's model: the agent has never acted upon the box before, therefore has never witnessed the effect, therefore has not acquired an affordance for the box.


Example 3: Sahin's model correctly predicts affordance
Sitability of broken chair
Our model: if the chair does not appear broken (i.e. a spring is broken on the inside) the agent would predict sitability based on knowledge of past situations. This model would not serve the agent well in this situation.
Sahin's model: There would exist no affordance of the chair because no effect has been observed yet. Once the agent performs the behavior of sitting in the chair, the observed effect would be the chair breaking. The agent would then acquire the knowledge that there exists no sitability affordance of the chair.
